# Plan: Fitur Kirim Pesan (User -> Admin Chat)

## Goal
User bisa kirim pesan pertanyaan via form kontak, dan admin bisa lihat/balas pesan tersebut di dashboard admin.

## Yang Perlu Dibuat/Diubah

### 1. Data Layer
- **Buat baru**: `data/messages.json` - array kosong `[]` sebagai penyimpanan pesan
- **Tambah type**: `Message` di `lib/types.ts`
  ```
  id, userId, userName, userEmail, subject, message, 
  status: "unread" | "read" | "replied", 
  adminReply?, createdAt, updatedAt
  ```

### 2. API Routes
- **Buat baru**: `app/api/messages/route.ts`
  - `POST` - user kirim pesan (validasi: name, email, subject, message)
  - `GET` - admin ambil semua pesan (hanya admin yang bisa)
  
- **Buat baru**: `app/api/messages/[id]/route.ts`
  - `GET` - ambil detail pesan
  - `PUT` - admin update status + balas pesan
  - `DELETE` - admin hapus pesan

### 3. Komponen Baru
- **Buat baru**: `components/home/ContactSection.tsx`
  - Form "Kirim Pesan" (nama, email, subjek, pesan)
  - Info kontak di samping (email, telepon, lokasi) - sesuai design yang user kasih
  - Submit ke `POST /api/messages`

### 4. Halaman Admin
- **Buat baru**: `app/(dashboard)/admin/pesan/page.tsx`
  - Daftar semua pesan (inbox style)
  - Badge status (unread/read/replied)
  - Klik untuk lihat detail + form balas
  - Delete pesan

### 5. Update Existing Files
- **Edit**: `components/layout/Sidebar.tsx`
  - Tambah item "Pesan" di `adminNav` array
  
- **Edit**: `app/page.tsx`
  - Import + render `ContactSection` sebelum `<Footer />`

## Verification
1. Buka halaman utama, scroll ke bawah - form kontak muncul
2. Isi form, klik "Kirim Pesan" - pesan terkirim
3. Login sebagai admin, lihat sidebar - menu "Pesan" muncul
4. Buka menu "Pesan" - inbox pesan terlihat
5. Klik pesan, balas - status berubah jadi "replied"
6. Coba hapus pesan - pesan hilang
